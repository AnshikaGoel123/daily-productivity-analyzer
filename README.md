# DAILY-PRODUCTIVITY-ANALYZER

A Flask-based web app that predicts productivity score using:

- Sleep Hours
- Focused Work Hours
- Focus Level (1-10)

## Tech Stack
- Python
- Flask
- HTML/CSS
- Scikit-learn

## Features
- Productivity score (0-100)
- Input validation
- Modular architecture
- Future ML integration

## Run Locally

pip install -r requirements.txt
python app.py
